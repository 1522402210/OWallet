## 总体介绍

OWallet是本体综合性桌面版钱包，支持标准的钱包管理、基于多重签名技术的共享钱包管理，同时将连接Ledger、Trezor多种硬件钱包。未来，OWallet将持续集成更加丰富的应用功能，为开发者提供智能合约编译、运行等综合性服务。

## 如何开始

1. 克隆repo

```
git clone https://github.com/ontio/OWallet.git
```

2. 安装依赖
推荐使用**yarn**

```
yarn install
```

3. 开发模式运行

```
yarn run dev
```

4. 打包

```
yarn run package
```

> 不要使用“npm run package”，否则windows下app会有问题。
